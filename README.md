v2 – переработанный калькулятор с одним полем ввода и кнопками цифр. 
Новые функции: 
- Ввод чисел через интерфейсные кнопки (0–9)
- Поддержка операций: +, -, *, /, ^
- Отображение результата в защищённом поле (readonly)
- Хранение первого числа и выбранной операции для последовательных вычислений
- Общая функция вычисления результата вместо отдельной функции для каждой операции
v2 – redesigned calculator with a single input field and number buttons.
New features:
- Input numbers via interface buttons (0–9)
- Supports operations: +, -, *, /, ^
- Displays results in a protected (readonly) field
- Stores the first number and selected operation for consecutive calculations
- Single result function instead of separate functions for each operation
